# refactoring
Support exercises for Refactoring classes


## Build
With [gradle](http://gradle.org) installed, run

    gradle --daemon build

If gradle is not installed, the provided `gradlew` helper scripts can be used.

On OS X and Linux run

    ./gradlew build

On Windows, run

    gradlew build
